## 7. Pane 3：リアルタイム計算結果

### 7.1 仕様どおりの計算ロジック（合意ベース）

**1. 総カウント台数（スーパークレジット加味）**

```
総カウント台数
  = ICE/HEV台数 + PHEV台数 + BEV（通常）台数
    + BEV（小型）台数 × 小型EVスーパークレジット係数
```

**2. 各車種の調整後排出量（燃料クレジット）**

```
調整後排出量 = Max(0, 個別WLTP − 燃料クレジット)   … ICE/HEV・PHEV
調整後排出量 = 0                                  … BEV系
```

**3. 自社フリート平均CO₂（加重平均）**

```
フリート平均CO₂
  = Σ(調整後排出量 × 販売台数) ÷ 総カウント台数
```

**4. クレジット加味後の実質排出量**

```
実質排出量 = フリート平均CO₂ − グリーン鉄クレジット
```

**5. 規制達成率（％）**

```
達成率 = (目標CO₂ ÷ 実質排出量) × 100
```

※実質排出量が0のときの扱いは実装時にガード必要（未決の細部）

**6. 目標との超過/猶予（g/km）**

```
超過g = 実質排出量 − 目標CO₂
```

（マイナス＝猶予）

**7. 想定総罰金額（€）**

```
罰金 = Max(0, 超過g) × 実際の総販売台数（スーパークレジット加味前） × 罰金単価
```

**表示要件（仕様）：**

- 達成率・超過g・総罰金を**大きな文字**
- 達成／未達で**緑／赤**の色分け

### 7.2 グリルで上乗せした計算要素

| 項目 | 合意内容 |
|------|----------|
| 目標CO₂ | **重量補正後の実効目標**を基本（問い21）。手入力上書き時はその値を使用 |
| 対象年 | **Pane 2で選んだ年**の目標・パラメータで計算 |
| 計算の置き場 | フロントでリアルタイム計算。**`lib/calculations.ts` に集約** |
| 罰金・達成率 | 上記仕様式をv1のコア。公式申告そのものではないが**公式寄り試算**として使う |
